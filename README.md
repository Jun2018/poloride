# Poloride 카메라 #

포토 프린터를 이용하여 사진을 뽑았을 때 폴라로이드 필름처럼 보이게 하는 폴라로이드 카메라 어플 “POLORIDE”



## 기능소개 ##
* 촬영 인터페이스를 카메라처럼 꾸미고 사진에 폴라로이드 프레임을 합친다.
* 촬영 직후에는 사진이 하얗게 보이며 일정 시간이 지나거나 핸드폰을 흔들면 깨끗한 사진을 출력한다.
* 갤러리에서 사진 선택 시 공유, 꾸미기, 사진 삭제를 할 수 있다.
* 꾸미기 : 사진에 필터, 드로우, 텍스트 작성, 날짜표시를 할 수 있다.
* 사진을 갤러리에서 선택하여 포토 프린터로 출력 할 수 있다.
